# Transcriptions rules

- superscript : ^ + letter   -> if two letter ^ letter ^other    etc. 
- unreadable : -[ + letters or interpretable (if readable, else "...") + ]-
- correction : [[ readable | correction]], if no readble : [[correction]]
- crossed out : *[word]*
- underline : _letters_
- new paragraph : ⁋
